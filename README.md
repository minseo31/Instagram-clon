## 개발 개요 📜
- SNS 플랫폼 중 하나인 모바일 환경에서의 Instagram의 클론 웹 서비스 입니다. 첫 풀스택 개발이며 NoSQL를 사용한 데이터 저장방식이 사용되었고 JWT 토큰 관리를 통한 계정보안이 구현되었습니다.
